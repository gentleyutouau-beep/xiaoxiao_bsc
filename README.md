This is a single-page landing site for the Xiaoxiao meme coin built with Next.js + Tailwind CSS.

Run locally:

1. npm install
2. npm run dev

Deploy to Vercel

- Import this repository at https://vercel.com/import and Vercel will auto-detect Next.js.
- Or deploy from the CLI:
  1. npm i -g vercel
  2. vercel login
  3. vercel --prod

Build & CI

- Build command: `npm run build`
- Output: Managed by Next.js on Vercel (no static output dir required)
